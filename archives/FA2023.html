p-lnl3d (Industry Project)

Partnered with San Diego Startup LNL3D Software development FOR 3D Printer SYNCHRONIZATION In this project, we will be developing source code for the company LNL3D in order to allow one 3D printer head to take over the other when the filament has run out. We will be utilizing Klipper firmware as parts of Python and we will also be testing it on a physical 3D printer unit. Klipper documentation Python language Member requirements: Python, willing to learn Klipper language.
