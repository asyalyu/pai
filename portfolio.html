Partnered with San Diego Startup LNL3D
Software development FOR 3D Printer SYNCHRONIZATION.
In this project, we developed source code for the company LNL3D to allow one 3D printer head to take over the other when the filament has run out.
We utilized Klipper firmware, Python, and tested it on a physical 3D printer unit.
Skills: Python, willingness to learn Klipper firmware.
